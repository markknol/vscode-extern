typedef InputBoxOptions

package vscode

Options to configure the behavior of the input box UI.

Properties

password:Null<Bool>

Set to true to show a password prompt that will not show the typed value.

placeHolder:Null<String>

An optional string to show as place holder in the input box to guide the user what to type.

prompt:Null<String>

The text to display underneath the input box.

validateInput:Null<String ‑> String>

An optional function that will be called to valide input and to give a hint to the user.

Parameters:

value

The current value of the input box.

Returns:

A human readable string which is presented as diagnostic message. Return undefined, null, or the empty string when 'value' is valid.

value:Null<String>

The value to prefill in the input box.
